# Frontend - GEM-R Portfolio

This directory contains the React Native / Expo frontend for the GEM-R Portfolio project.

## Scripts

-   `npm install`: Install dependencies.
-   `npx expo start`: Start the development server.
-   `npm run android`: Run on Android emulator.
-   `npm run ios`: Run on iOS simulator.

## Project Structure

-   `app/`: Application source code and screens (expo-router).
-   `components/`: Reusable UI components.
-   `assets/`: Images and fonts.

Refer to the root [README.md](../README.md) for full project details.
